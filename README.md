# crypto-exchange-simulator

## Project setup
```
npm install
npm install -g @vue/cli
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run the production app (require npm run build)
```
npm run start
```

### Lints and fixes files
```
npm run lint
```

### Run your unit tests
```
npm run test:unit
```
